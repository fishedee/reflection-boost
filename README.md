# reflection-boost

Java下的反射加强工具库，功能包括：

* 泛型计算，动态计算泛型类下的方法和字段的实际参数是什么

## 安装

```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>

<dependency>
    <groupId>com.github.fishedee</groupId>
    <artifactId>reflection-boost</artifactId>
    <version>1.0</version>
</dependency>
```

在项目的pom.xml加入以上配置即可
